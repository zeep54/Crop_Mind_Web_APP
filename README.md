# Crop_Mind_Web_APP
A short website that is connected to Crop Mind firmware to get data from both the sd card and the wireless!!

## data/latest.json

This file is **not** edited by hand and **not** generated by any build step in this repo. It's overwritten periodically by the separate Pico 2 W firmware, which pushes to it directly via the GitHub Contents API using a fine-grained token scoped only to this repo. It exists so the dashboard can show a recent reading from anywhere, not just when the browser is on the same network as the device — see the "GET LATEST (from GitHub)" call in [index.html](index.html). Don't add a GitHub Action or other automation that touches this file; it's owned entirely by the external device.
